📖 Panduan Penggunaan UMUMKU

1
Login ke aplikasi
Gunakan akun yang sudah diberikan. Contoh: admin / admin123 atau sesuai bagian Anda.
2
⚠️ Pilih Divisi & Bulan di Dashboard (WAJIB pertama!)
Setelah login, langsung isi Divisi/Bagian dan Bulan di kartu "Atur Sesi Permintaan". Data tidak akan tersimpan dengan benar jika langkah ini dilewati.
3
Pilih kategori: ATK atau Computer Supply (CS)
Klik tombol "✏️ Mulai Pilih ATK" atau "🖨️ Mulai Pilih CS" dari Dashboard, atau klik menu di sidebar kiri.
4
Pilih barang dan masukkan jumlah
Klik kartu barang → isi jumlah (volume) → klik Tambahkan. Ulangi untuk semua barang yang dibutuhkan.
5
Cek di Rekap Permintaan
Buka menu Rekap Permintaan untuk melihat semua pilihan. Anda bisa edit volume atau koreksi langsung di tabel.
6
Kirim ke Google Spreadsheet atau Download CSV
Klik "↑ Kirim ke Sheet" untuk langsung kirim ke Google Spreadsheet (butuh setup Apps Script dulu), atau "⬇ CSV" untuk download file.
⚠️ Mengapa gambar tidak muncul dari Google Drive?
Google Drive "uc?export=view" sering diblok oleh browser karena CORS. Aplikasi ini sudah menggunakan Thumbnail API Google yang jauh lebih stabil untuk file publik.
1
Pastikan gambar sudah di-share "Anyone with the link"
Klik kanan file di Drive → Share → ubah ke Anyone with the link → Viewer → Done. Ini wajib agar gambar bisa tampil.
2
Paste URL Drive biasa — otomatis dikonversi!
Cukup paste URL Drive biasa saat tambah/edit barang:
https://drive.google.com/file/d/1ZiCBd3TOV6nUWlUABfMZ7Paq0jvHGDUE/view
Aplikasi otomatis mengubah ke format Thumbnail API:
https://drive.google.com/thumbnail?id=FILE_ID&sz=s400
✅ Format ini bekerja tanpa CORS issue di semua browser.
3
Gambar masih tidak muncul? Tambah manual di app.js
Buka file app.js, cari array DEFAULT_PRODUCTS, lalu isi kolom gambar langsung dengan URL thumbnail:
gambar: "https://drive.google.com/thumbnail?id=FILE_ID_ANDA&sz=s400"
Ganti FILE_ID_ANDA dengan ID file dari URL Drive Anda.
4
Alternatif paling stabil: ImgBB
Upload ke imgbb.com → salin "Direct Link" (berakhiran .jpg/.png) → paste ke kolom gambar. URL ini tidak punya masalah CORS sama sekali.
⚡ 2 Spreadsheet Output!
Data dikirim ke 2 spreadsheet sekaligus: (1) Per Divisi — tiap divisi punya tabel sendiri, (2) Semua Divisi — semua divisi digabung dalam 1 sheet bulan. Format sama persis dengan Excel Perhutani.
1
Buat 2 Google Spreadsheet baru
Buat 2 spreadsheet kosong di Google Drive. Contoh nama: UMUMKU — Per Divisi dan UMUMKU — Semua Divisi. Sheet di dalam akan dibuat otomatis.
2
Ambil ID dari kedua spreadsheet
Dari URL masing-masing spreadsheet, ambil bagian ID-nya:
https://docs.google.com/spreadsheets/d/ID_INI/edit
3
Buka Apps Script dari salah satu spreadsheet
Klik menu Extensions → Apps Script. Hapus semua kode yang ada, paste kode dari file Code.gs yang sudah didownload.
4
Isi 2 ID spreadsheet di bagian atas kode
Di baris paling atas Code.gs, ganti 2 nilai ini:
const ID_SPREADSHEET_PER_DIVISI = "ID_SPREADSHEET_1";
const ID_SPREADSHEET_SEMUA_DIVISI = "ID_SPREADSHEET_2";
5
Deploy sebagai Web App
Deploy → New Deployment → Web App
Execute as: Me | Who has access: Anyone
→ Klik Deploy → salin URL yang muncul.
6
Tempel URL ke file app.js
Buka app.js, cari:
const APPS_SCRIPT_URL = "GANTI_DENGAN_URL...";
Ganti dengan URL deployment tadi. Simpan.
Output dari web sama persis dengan format Excel Perhutani — 3 baris judul, header kolom, data per divisi dengan subtotal, dan grand total di bawah.
